## Search Scope Exclusions

Always exclude these paths from Glob and Grep searches — they are not source-of-truth:
- `.auto-claude/worktrees/` — Claude Code worktree snapshots that mirror the repo; searching them returns duplicate, stale results
- `kubernetes/overlays/homelab/infrastructure/kubevirt/resources/kubevirt-operator.yaml` — vendor-generated (7,669 lines)
- `kubernetes/overlays/homelab/infrastructure/kubevirt-cdi/resources/cdi-operator.yaml` — vendor-generated (5,486 lines)
- `kubernetes/bootstrap/cilium/cilium.yaml` — generated by `make cilium-bootstrap`, never hand-edited

When searching Kubernetes manifests, prefer `kubernetes/overlays/homelab/` over `kubernetes/` root to avoid base/overlay ambiguity.

When searching for an app's configuration, start at `kubernetes/overlays/homelab/infrastructure/<app-name>/`.
